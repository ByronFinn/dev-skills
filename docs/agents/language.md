# 文档语言 / Documentation Language

工程技能为本仓库撰写**面向人类读者输出**时使用的语言。

## 语言

**简体中文**

所有技能产出的文档内容——PRD（`docs/prd/`）、ADR（`docs/adr/`）、`CONTEXT.md`、Issue 标题与正文、评审报告以及评论——均使用简体中文书写。

## 范围

- **适用**：文档和 Issue 中面向人类读者的行文。
- **不适用**：代码、标识符、文件名、CLI 命令以及配置值。这些遵循代码规范，不受本设置影响。

## 当技能需要"撰写输出"时

按照上述语言撰写行文。不要擅自切换到对话所使用的语言——团队阅读的是配置中指定的语言，可能和触发技能的人不同。

如果某个产物确实需要第二种语言（例如双语发布说明），产出技能可以偏离配置，但配置语言始终是主要语言。

## 如果本文件缺失

回退到用户的输入语言，并询问团队偏好的语言以便记录在此。不要在每个会话中都靠猜测。
